## Before reporting a bug

Please make sure that:

- You are using the latest available version of Pocket Alchemy.
- You have checked the existing issues.
- You can reproduce the problem consistently.
- You know your Minecraft, NeoForge, and Pocket Alchemy versions.
- You can provide your `latest.log`.
- You have tested whether another installed mod may be causing the problem.

## Bug reports

A useful bug report should include:

- Minecraft version
- NeoForge version
- Pocket Alchemy version
- Steps required to reproduce the problem
- Expected behavior
- Actual behavior
- Relevant installed mods
- `logs/latest.log`
- Crash report, if one was generated
- Screenshots or video, when useful

The `latest.log` file is required even when the game does not crash. It can contain mod conflicts, warnings, loading errors, and other information needed to identify the cause.

## Ideas

Ideas should explain:

- What should be added or changed
- How the proposed mechanic should work
- What problem it solves
- Why it fits Pocket Alchemy
- Whether there are alternative implementations

Submitting an idea does not guarantee that it will be implemented.
